# -codealpha_stock-portfolio-tracker
📈 Stock Portfolio Tracker using Python

📌 Project Overview

The Stock Portfolio Tracker is a Python-based application that helps users manage and track their stock investments. It allows users to add stocks, record the number of shares owned, calculate portfolio value, and monitor investment performance efficiently.

🚀 Features

- Add and manage stock holdings
- Store stock names and quantities
- Calculate total portfolio value
- Display individual stock details
- Simple and user-friendly interface
- Error handling for invalid inputs
- Easy to customize and extend

🛠️ Technologies Used

- Python 3
- Dictionaries
- Functions
- Loops
- Conditional Statements
- File Handling (Optional)

📂 Project Structure

stock-portfolio-tracker/
│
├── stock_tracker.py
├── README.md
└── portfolio_data.txt (optional)

▶️ How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Run the following command:

python stock_tracker.py
